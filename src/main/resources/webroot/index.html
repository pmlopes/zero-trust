  1. A request from the user browser will arrive at the server secured connection using HTTPS
  2. If user is not authenticated, user is redirected to Google to assert the identity
  3. Google will return an access_token + id_token
  4. Request will go to the microservice including id_token
  5. Microservice will attest id_token and execute
  • AuthZ will be performed on all nodes (and require a fresh token (5s))
